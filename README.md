> Tampermonkey userscript that visualizes Claude.ai conversation 
> branches as an interactive tree — jump to any message, 
> collapse subtrees, label nodes, and more.

![screenshot](screenshots/tree-view.png)

## Features

### Tree View
Displays user messages as a hierarchical tree. 
The active branch is highlighted.

![tree view](screenshots/tree-view.png)

### Node Jump
Click any node to scroll to that message 
with a brief highlight animation.

![node jump](screenshots/node-jump.gif)

## Installation

1. Install [Tampermonkey](https://www.tampermonkey.net/) 
   for your browser
2. **[Click here to install the script]
   (https://raw.githubusercontent.com/Sc5N6A9L/claude-branch-tree-viewer/main/branch-tree-viewer.user.js)**
3. Click "Install" on the Tampermonkey dialog
4. Open [claude.ai](https://claude.ai) — the tree icon 
   appears in the toolbar
